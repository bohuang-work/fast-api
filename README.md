# Fast API

RESTful-api implemented using Fast-API framework:

- hexagon structure
- HTTP methods: CRUD
- PostgreSQL

## pre installation

- Docker
- Python >= 3.8

## setup

1. run docker compose to start API and postgresDB

```
   cd backend
   docker docker-compose up
```

2. open:
   http://127.0.0.1:8080/docs
